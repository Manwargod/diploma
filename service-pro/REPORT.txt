1. ОПИСАНИЕ ПРОЕКТА
================================================================================
ServiceTap.kz - это многофункциональная платформа для технического обслуживания,
ремонта компьютеров и ноутбуков, а также продажи компьютерных комплектующих
в Казахстане. Платформа включает сервис центры, маркетплейс и конфигуратор ПК.

Технологический стек:
- Frontend: React.js
- Маршрутизация: React Router v6
- UI компоненты: Tailwind CSS
- Иконки: Lucide React
- 3D визуализация: Three.js
- Поддержка языков: RU, KZ, EN

3. ОСНОВНОЙ ФУНКЦИОНАЛ
================================================================================

3.1 ГЛАВНАЯ СТРАНИЦА (Home.js)
[МЕСТО ДЛЯ СКРИНШОТА 1: Главная страница с героем]
Функционал:
- Герой секция с призывом к действию
- Навигация по основным сервисам
- Информация о компании
- Контактная информация

3.2 СТРАНИЦА РЕМОНТА (Repair.js)
[МЕСТО ДЛЯ СКРИНШОТА 2: Страница ремонта]
Функционал:
- Категории услуг ремонта (смартфоны, ноутбуки, ПК, консоли)
- Форма заказа ремонта
- Отправка заявки с сохранением данных
- Успешное подтверждение заказа

3.3 КОНФИГУРАТОР ПК (Build.js)
[МЕСТО ДЛЯ СКРИНШОТА 3: Конфигуратор ПК]
Функционал:
- 3D визуализация компьютера (интерактивная модель вращается)
- Три готовых пресета конфигураций:
  * Бюджетная (250 000 ₸ - 350 000 ₸)
  * Оптимальная (550 000 ₸ - 750 000 ₸)
  * Ультимативная (1 200 000 ₸ - 1 500 000 ₸)
- Выпадающие меню для выбора комплектующих:
  * Процессор (CPU)
  * Видеокарта (GPU)
  * Материнская плата
  * Оперативная память (RAM)
  * Хранилище (SSD)
  * Блок питания (PSU)
- Каждый компонент имеет опцию "Нет" для отказа от покупки
- Фильтрация комплектующих по кастомному бюджету
- Информация о цене каждого компонента
- Выбор адреса доставки
- Отправка заявки на сборку
- Страница подтверждения с выбором сервис центра для гарантии

[МЕСТО ДЛЯ СКРИНШОТА 4: Визуализация ПК]
[МЕСТО ДЛЯ СКРИНШОТА 5: Выбор комплектующих]
[МЕСТО ДЛЯ СКРИНШОТА 6: Выбор сервис центра]

3.4 МАРКЕТПЛЕЙС (Market.js)
[МЕСТО ДЛЯ СКРИНШОТА 7: Маркет с товарами]
Функционал:
- Поиск товаров в реальном времени
- Фильтрация по категориям:
  * Смартфоны
  * Ноутбуки
  * Компьютерное железо
- Красивые карточки товаров с:
  * Изображением (плейсхолдер)
  * Рейтингом и количеством отзывов
  * Названием и спецификациями
  * Информацией о наличии
  * Ценой
  * Кнопкой добавления в корзину
- Кликабельные карточки для перехода на подробную страницу товара

3.5 СТРАНИЦА ТОВАРА (ProductDetail.js)
[МЕСТО ДЛЯ СКРИНШОТА 8: Подробная страница товара]
Функционал:
- Полная информация о товаре
- Большое изображение товара
- Рейтинг со звёздами
- Полное описание товара
- Характеристики
- Список основных возможностей с галочками
- Кнопка "Добавить в корзину"
- Информация о наличии в городах
- Рекомендованные товары похожей категории
- Кнопка "Вернуться в маркет"

3.6 ПРОВЕРКА ГАРАНТИИ (Warranty.js)
[МЕСТО ДЛЯ СКРИНШОТА 9: Страница проверки гарантии]
Функционал:
- Ввод серийного номера (S/N) товара
- Проверка статуса гарантии
- Отображение информации о гарантии
- Контактная информация сервис центра

3.7 ЛИЧНЫЙ КАБИНЕТ (Dashboard.js)
[МЕСТО ДЛЯ СКРИНШОТА 10: Личный кабинет]
Функционал:
- История заказов (ремонт и сборки)
- Отслеживание статуса текущих заказов
- Информация о гарантии
- Профиль пользователя

5. ДИЗАЙН И СТИЛИЗАЦИЯ
================================================================================

Цветовая схема:
- Основной цвет: #00f2ff (голубой, неоновый)
- Вторичный цвет: #7000ff (фиолетовый)
- Светлый режим: белый фон с чёрным текстом
- Тёмный режим: тёмный фон (#0f0f12) с белым текстом
- Успешные операции: #00c853 (зелёный)
- Ошибки: красный

Компоненты используют:
- Закруглённые углы (border-radius: 2rem, 2.5rem, 3rem)
- Тени для глубины
- Градиенты для кнопок и эффектов
- Плавные переходы (transition)
- Hover эффекты

ServiceTap.kz - это амбициозный проект, который объединяет услуги ремонта,
продажи техники и конфигурации ПК в единую платформу. Текущая версия 
демонстрирует основную функциональность, и имеет четкий план развития на
следующие 18 месяцев.

Проект готов к:
✓ Демонстрации инвесторам
✓ Тестированию пользователями
✓ Постепенному масштабированию

Следующие этапы будут сосредоточены на создании надежного бэкэнда,
расширении функциональности мобильной версии и установлении партнёрств
с ключевыми игроками рынка.
