### 1. **Система сравнения товаров** ⚖️

**Файл:** `src/components/ComparisonTable.js` (новый)

**Функциональность:**

- ✅ Детальная таблица со всеми характеристиками
- ✅ Коллапсируемые категории спецификаций
- ✅ Визуальное выделение преимуществ
- ✅ Кнопки "Купить" для каждого товара
- ✅ Модальное окно на весь экран
- ✅ Поддержка светлого/тёмного режима

**Примеры использования:**
```javascript

  products={compareList}      // Массив товаров для сравнения
  isDark={isDark}             // Режим
  t={translations}            // Переводы

/>
```

## 🚀 Как запустить и протестировать

### 1. Установка зависимостей:
```bash
cd service-pro
npm install
```

### 2. Запуск разработки:
```bash
npm start
```

### 3. Проверка функций:

**Сравнение товаров:**
- Наведите курсор на карточку товара
- Кликните на иконку ⚖️
- Выберите 2-4 товара
- Нажмите кнопку "Сравнить"

**Фильтрация:**
- Используйте Range Slider для цены
- Выберите бренды из списка
- Выберите рейтинг
- Товары автоматически обновляются

**Добавление в корзину:**
- Наведите курсор на карточку
- Нажмите иконку 🛒
- Произойдёт событие addToCart (см. консоль)
