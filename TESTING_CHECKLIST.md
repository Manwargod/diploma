- Учитывается регистр (case-insensitive)
- Может быть комбинирован с фильтрами

### ✅ Фильтрация по категориям
1. Откройте Market
2. Нажимайте на различные категории (Все, Смартфоны, Ноутбуки, Железо)
3. Товары должны обновляться

**Ожидаемый результат:**
- Только товары из выбранной категории показываются
- "Все" показывает все товары
- Активная кнопка выделена кианом

### ✅ Фильтрация по цене
1. Откройте Market
2. Перемещайте слайдер цены влево-вправо
3. Товары должны фильтроваться в реальном времени

**Ожидаемый результат:**
- Товары дороже выбранной цены скрываются
- Изменение происходит без задержек
- Отображается текущее значение (K ₸)

### ✅ Фильтрация по бренду
1. Откройте Market
2. Выберите несколько брендов (например, Apple и Samsung)
3. Товары должны обновиться (показать товары только этих брендов)
4. Снимите выбор
5. Товары должны вернуться в исходное состояние

**Ожидаемый результат:**
- Множественный выбор работает
- Можно выбирать/отменять выбор
- Список обновляется мгновенно

### ✅ Сравнение товаров
1. Откройте Market
2. Наведите курсор на карточку товара
3. Нажмите иконку ⚖️
4. Товар добавится в список сравнения (видно сверху)
5. Повторите для 2-4 товаров
6. Нажмите кнопку "Сравнить"
7. Откроется модальное окно с таблицей сравнения
8. Попробуйте коллапсировать/раскрывать категории
9. Нажмите X для закрытия

**Ожидаемый результат:**
- До 4 товаров можно выбрать для сравнения
- Sticky панель показывает выбранные товары
- Таблица открывается в модальном окне
- Все характеристики видны
- Модальное окно закрывается

### ✅ Добавление в корзину
1. Откройте Market
2. Наведите курсор на карточку товара
3. Нажмите иконку 🛒
4. Посмотрите в консоль браузера (F12 → Console)
5. Должно быть сообщение об добавлении товара

**Ожидаемый результат:**

## 📞 Если есть проблемы:

1. **Посмотрите консоль браузера** (F12 → Console)
2. **Проверьте Network tab** на ошибки загрузки
3. **Прочитайте сообщение об ошибке** (обычно указывает на проблему)
4. **Проверьте пути импортов** в компонентах
5. **Убедитесь, что все файлы созданы** в правильных местах
6. **Перезагрузите браузер** (Ctrl+F5 - hard refresh)
7. **Удалите node_modules и переустановите** (если очень плохо)

```bash
# Если сильно подвис
rm -r node_modules
npm install
npm start
```
