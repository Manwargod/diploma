## 🎯 Особенности реализации

### Многоязычность
- Для каждого текстового элемента используется условный оператор:
  ```javascript
  {t?.buildBtn === 'Build PC' ? 'English Text' : t?.buildBtn === 'ПК жинау' ? 'Kazakh Text' : 'Russian Text'}
  ```

### Универсальность
- Переключение языка работает глобально через `setLang()` в App.js
- Все компоненты автоматически обновляются при смене языка
- Нет необходимости в перезагрузке страницы
